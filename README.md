# Expense Tracker App

A full-stack expense tracking Progressive Web App (PWA)
that works on mobile and desktop.

## Tech Stack
- Frontend: React (PWA)
- Backend: FastAPI (Python)
- Database: PostgreSQL
- Auth: JWT

## Features (Planned)
- Add daily expenses
- Category-wise tracking
- Monthly summaries
- Charts & insights
- Mobile + Desktop support
